# Glacier

macOS 메뉴바 아이템을 숨겨주는 경량 유틸리티 — 접근성 권한 불필요.

[Ice](https://github.com/jordanbaird/Ice)와 같은 확장 트릭을 사용하지만, 의존성 없이 Swift ~200줄로 구현되었습니다.

[English](README.md)

## 작동 원리

Glacier는 메뉴바에 보이지 않는 구분자를 배치합니다. 구분자를 확장하면 왼쪽 아이템들이 화면 밖으로 밀려납니다.

두 개의 마커로 세 구간을 나눕니다:

```
[항상 숨김] ◆ [숨김] ● [보이는 것]
```

| 마커 | 역할 |
|------|------|
| **●** | 클릭 대상 — 숨김 섹션 토글 |
| **◆** | 경계 표시 — "숨김"과 "항상 숨김" 구분 |

## 사용법

| 동작 | 결과 |
|------|------|
| **● 클릭** | 숨김 섹션 토글 |
| **Option + ● 클릭** | 항상 숨김 섹션 토글 |
| **● 우클릭** | 종료 메뉴 |
| **다른 곳 클릭** | 전부 숨김 |

**Cmd + 드래그**로 ●와 ◆ 마커를 이동하여 각 섹션에 속하는 아이템을 조정할 수 있습니다.

## 상태

| 상태 | 화면에 보이는 것 |
|------|-----------------|
| 전부 숨김 (기본) | `●  [보이는 아이템들]` |
| 부분 표시 | `◆  [숨김 아이템들]  ●  [보이는 아이템들]` |
| 전부 표시 | `[항상 숨김]  ◆  [숨김 아이템들]  ●  [보이는 아이템들]` |

## 설치

### 다운로드

[Releases](../../releases)에서 최신 `.zip`을 다운로드하고, 압축 해제 후 `Glacier.app`을 `/Applications`로 드래그하세요.

### 소스에서 빌드

Xcode 16+ 및 macOS 15+ 필요.

```bash
git clone https://github.com/junuMoon/Glacier.git
cd Glacier
xcodebuild -scheme Glacier -configuration Release build
```

## 요구사항

- macOS 15.0 (Sequoia) 이상
- 접근성 권한 불필요
